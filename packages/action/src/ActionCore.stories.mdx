# ActionCode

export const Template = (args) => <ActionCore {...args} />;

`ActionCore` is a small round badge that can act as a link or a button.

## Link

To act as a link, an `ActionCode` requires the `href` prop.

## Button

If you want a button-like behaviour, just add an `onClick` event-handler. The event parameter is of type:

```js
{
  id: 'idPassedAsProp';
}
```
