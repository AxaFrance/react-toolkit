# Title

export const Template = (args) => <Title {...args} />;

The title component represents a title inside your application.

By default, without changing the `className` and `classModifier` props, the title will have an `af-title` css class.  
If you set the `className` property, the `af-title` css class will be removed and only the class set in `className` will be set on the html element.  
If you set the `classModifier` property, the `af-title` css class will be kept but enriched with the value of the `classModifier` property.

Don't forget to import the component css file, so that your title is styled thanks to the `af-title` css class.
```javascript
import '@axa-fr/react-toolkit-title/dist/af-title.css';
```
