# Table with paging

You have the option to have a paging below your table with the `Paging` component.  
If you choose so, you will have to install the `select` component as well.

```shell script
npm i @axa-fr/react-toolkit-form-input-select
npm i @axa-fr/react-toolkit-form-core
```

Don't forget to import the components css files, so that your table and the paging have the correct style.

```javascript
import '@axa-fr/react-toolkit-table/dist/af-table.css';
import '@axa-fr/react-toolkit-table/dist/Paging/af-paging.css';
import '@axa-fr/react-toolkit-table/dist/Pager/af-pager.css';
import '@axa-fr/react-toolkit-form-input-select/dist/af-select.css';
import '@axa-fr/react-toolkit-form-core/dist/af-form.css';
```
