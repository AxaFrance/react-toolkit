The modal default is the component to use if you want to fully configure your modal.

With the `Modal.HeaderBase` you can configure the header of the modal. With the `Modal.Body` you can configure the body of the modal. And with the `Modal.Footer` you can configure the footer of the modal.

Don't forget to load the css file of the component.

```javascript
import '@axa-fr/react-toolkit-modal-default/dist/af-modal.css';
```
