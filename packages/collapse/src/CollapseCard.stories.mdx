The collapse package exports a `CollapseCard` component that allows you to show some content that will be display and hide when you click on an header.  
The header is defined inside the `CollapseCard.Header` and the content that will be shown after clicking on that header is defined inside the `CollapseCard.Body` component.  
The `onToggle` allows you to define an action to execute when the collapse is open or close. An event is sent containing an generated `id`, the `index` of the collapse that is being used (0 if there's just one collapse element), and an `isOpen` boolean property to tell if the action closed or opened the collapse element.

Don't forget to load the css file of the component.

```javascript
import '@axa-fr/react-toolkit-collapse/dist/af-accordion.css';
```
