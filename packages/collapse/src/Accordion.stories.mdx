The collapse package exports a `Accordion` component that allows you to show multiple collapse elements (for documentation about the `CollapseCard` component refer to [this](?path=/docs/components-low-level-collapse--collapse)).  
You can specify if you want multiple collapse to open at the same time inside your accordion or just one at a time with `onlyOne` boolean property. By default, it is set to false which allows you to open more than more.

Don't forget to load the css file of the component.

```javascript
import '@axa-fr/react-toolkit-collapse/dist/af-accordion.css';
```
