# Help

export const Template = ({ children, ...args }) => (
  <Help {...args}>{children}</Help>
);

The help package exports a `Help` component that allows you to display an help message on top of an information button.  
That additionnal information will be shown either by hovering the main message or by clicking on it, depending on the `mode` property you are setting. For that, use the `PopoverModes` enumeration that is being install with the help package.  
By default, the `mode` property is set on `click`.

```javascript
PopoverModes.click, PopoverModes.over;
```

Also, you can set wherever you want that additionnal information to be shown on the screen with the `placement` property. Again, use the `PopoverPlacements` provided with the installation of the package.
By default, the `placement` property is set on `right`.

```javascript
PopoverPlacements.bottom,
PopoverPlacements.left,
PopoverPlacements.right,
PopoverPlacements.top,
```

Finally, the additionnal information will set with the `children` property. It is of `ReactNode` type so that you can display a string or a more elaborate content with HTML.

Don't forget to load the css file of the component.

```javascript
import '@axa-fr/react-toolkit-popover/dist/af-popover.css';
import '@axa-fr/react-toolkit-popover/dist/af-help-custom.css';
import '@axa-fr/react-toolkit-button/dist/af-button.css';
```
