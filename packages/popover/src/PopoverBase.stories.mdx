The PopoverBase is the base component of the popover package. It allows to interact directly and with more depth with the component (defining custom functions when the user enters or leaves the popover html block for example).

Don't forget to load the css file of the component.

```javascript
import '@axa-fr/react-toolkit-popover/dist/af-popover.css';
```
