The popover package provides a way to pop content over another one when hovering or clicking that html block.
The base component to use is the `Popover` component.
The content that is always visible on the screen is defined inside a `Popover.Over`, while the content that will pop after an user event is defined inside a `Popover.Pop`.
The way the extra content is shown on the page is defined by the `mode` property by using `PopoverModes.over` or `PopoverModes.click`.
You can also defined the target placement of that extra content with the `placement` property (for that you can use the `PopoverPlacements` enumeration).

Don't forget to load the css file of the component.

```javascript
import '@axa-fr/react-toolkit-popover/dist/af-popover.css';
```
