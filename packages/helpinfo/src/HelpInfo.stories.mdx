# HelpInfo

export const Template = ({ children, ...args }) => (
  <HelpInfo {...args}>{children}</HelpInfo>
);

The helpinfo package provides the same comportement as the help package (for more documentation, click [here](?path=/docs/components-high-level-help--text)) with the difference that you can show additionnal information hover any html content.  
This is made possible with the `children` property that will show the html elements you want on the page and the `content` property will set the additionnal property you want to show.  
On top of that, an `isDisabled` property has been added to disable the display of additionnal information by hovering or clicking the html content.

Don't forget to load the css file of the component.

```javascript
import '@axa-fr/react-toolkit-popover/dist/af-popover.css';
import '@axa-fr/react-toolkit-popover/dist/af-help-info.css';
```
